# yue.css

Style for readable content.

## Installation

Install with [component(1)](http://component.io):

    $ component install lepture/yue.css

## API

Style element under the `.yue` class.

```css
<div class="yue">
    <h1>Heading</h1>
    <p>Paragraph of contents...</p>
</div>
```
